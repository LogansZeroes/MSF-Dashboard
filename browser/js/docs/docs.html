This dashboard is meant to be viewed on a computer at a central location

  • begin by going to "Alerts"

  • you will need to enter a valid email and temperature range for the measurements

  • clicking the image will take you to "Home", which will show you the real-time chart

  • "Data" will show you the last 500 dweets

  • "Latest" will show you the last dweet

  • "Users" will show you a list of all registered users
